## functionAgent

### Description

It allows developers to implement the agent function within the graph itself.

### Samples

#### inputs

```json

[
          {
                    "model": "Model 3",
                    "maker": "Tesla",
                    "range": 300,
                    "price": 35000
          }
]

````

#### params

```json

{}

````

#### result

```json

"A Tesla Model 3 has the range of 300 miles. It costs $35000."

````
